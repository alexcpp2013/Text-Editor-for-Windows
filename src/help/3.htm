Аналіз структури програми і даних

 

Проект програми складається з наступних файлів:

- SEdit.cpp – головний модуль проекту, який містить точку входу в програму;

- еdit.cpp – модуль головної форми програми, який містить обробники подій візуальних компонентів, допоміжні функції та методи.

Більшість функцій, що виконуються програмою, оформлені у вигляді обробників подій від візуальних компонентів управління.

Програма призначена для виконання на персональних IBM PC сумісних комп’ютерах під керуванням операційних систем сімейства Windows.

Запуск програми здійснюється через завантаження виконавчого модуля SEdit.exe в операційній системі сімейства Windows активізацією піктограми програми подвійним щигликом миші або натисненням клавіші ENTER на клавіатурі.

Вхідна точка програми розміщена у головному модулі проекту – SEdit.cpp – і оформлена у вигляді функції WinMain:

#include <vcl.h>

#pragma hdrstop

//---------------------------------------------------------------------------

USEFORM("edit.cpp", Form1);

USEFORM("Unit2.cpp", AboutBox);

//---------------------------------------------------------------------------

WINAPI WinMain(HINSTANCE, HINSTANCE, LPSTR, int)

{

        try

        {

                 Application->Initialize();

                 Application->CreateForm(__classid(TForm1), &Form1);

                 Application->CreateForm(__classid(TAboutBox), &AboutBox);

                 Application->Run();

        }

        catch (Exception &exception)

        {

                 Application->ShowException(&exception);

        }

        catch (...)

        {

                 try

                 {

                         throw Exception("");

                 }

                 catch (Exception &exception)

                 {

                         Application->ShowException(&exception);

                 }

        }

        return 0;

}

Функція WinMain здійснює ініціалізацію і завантаження вікна програми забезпечує обробку системних помилок (виключень).

Для роботи з програмою необхідно курсор миші поставити на піктограму додатка і подвійним щигликом запустити її або одним щигликом миші виділити іконку програми і натиснувши клавішу Enter на клавіатурі запустити програму, також можна використовувати інші способи виклику додатку в операційних системах сімейства Windows.

Для відкриття файла необхідно використати меню Файл/Открыть.

Для створеня нового документа необхідно вибрати пункт меню Файл/Создать або натиснути кнопку на панелі.

Для збереження файлів можна використовувати відповідні пункти меню Файл або клавиші на панелі.

Меню Правка містить функції для вставки тексту (або просто натиснути Ctrl+V) і копіювання (або просто натиснути Ctrl+C).

Для змінення кольору фону або шрифта необхідно вибрати необхідні пункти у меню Формат.

Програма містить допоміжні матеріали для полегшення роботи користувача, а саме, файл довідки.

Короткі відомості про програмний продукт можна отримати викликавши пункт контекстного меню Справка/О программе…
